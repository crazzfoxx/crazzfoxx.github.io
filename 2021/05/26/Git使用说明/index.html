Git使用说明

发表于 2021-05-26 更新于 2021-05-28

github无法访问

修改C:\Windows\System32\drivers\etc\hosts（MAC用终端修改：sudo vim /etc/hosts），追加下面两行：

github.com从 https://github.com.ipaddress.com/ 中找
github.global.ssl.fastly.net从 https://fastly.net.ipaddress.com/github.global.ssl.fastly.net 中找

修改git默认编辑器

设置默认编辑器为notepad++
